# interview-iac-repo

## GitHub Infrastructure as Code using CDKTF

This project demonstrates managing GitHub resources using CDK for Terraform (CDKTF) with TypeScript.

### Features
- GitHub repository creation
- Branch protection on main branch
- GitHub Actions secret management
- Terraform outputs

### Prerequisites
- Node.js
- Terraform
- CDKTF
- GitHub Personal Access Token

### How to Deploy
1. Export `GITHUB_TOKEN`
2. Install dependencies: `npm install`
3. Deploy: `cdktf deploy`
